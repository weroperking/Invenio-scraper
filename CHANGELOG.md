## Frequently Asked Questions

### How do I know what version I'm using?

Check `package.json` or run:

```bash
pnpm list moviebox-js-sdk
```

### Where can I find the latest release?

Visit the [GitHub Releases](https://github.com/pythonvista/moviebox-js-sdk/releases) page.

### How do I report a bug?

Open an issue on [GitHub Issues](https://github.com/pythonvista/moviebox-js-sdk/issues) with:
- Description of the issue
- Steps to reproduce
- Expected vs actual behavior
- Environment details
